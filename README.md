## URL Shorter
A simple URL shortener built with Node.js ,Express and MongoDB in Backend and React in Frontend. This project allows users to create shortened URLs that redirect to the original long URLs.

### How to Run the Project
1. Install dependencies:
   ```bash
   cd backend
   npm install # or pnpm install
   cd ../frontend
   npm install # or pnpm install
2. Start the development server:
   ```bash
   npm run dev # on frontend
   node server.js # on backend
3. Open your browser and navigate to `http://localhost:5173` to access the frontend of the URL shortener.
